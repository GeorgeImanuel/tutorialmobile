ü§ñ Android Studio (Kotlin)

1Download Android Studio

Kunjungi website resmi Android Studio di https://developer.android.com/studio dan download versi terbaru.

2Instalasi Android Studio

Jalankan file installer yang telah didownload dan ikuti wizard instalasi. Pastikan untuk menginstall Android SDK dan Android Virtual Device.

3Konfigurasi Awal

Setelah instalasi selesai, buka Android Studio dan lakukan konfigurasi awal. Pilih tema yang diinginkan dan download komponen yang diperlukan.

4Membuat Project Baru

Klik "New Project" dan pilih template "Empty Activity". Isi detail project:

  • Name: HelloWorld
  • Package name: com.example.helloworld
  • Language: Kotlin
  • Minimum SDK: API 21

5Project HelloWorld Berhasil Dibuat

Setelah project selesai dibuat, Anda akan melihat struktur project di sebelah kiri dan file MainActivity.kt di editor.

üí° Catatan: Pastikan komputer Anda memiliki minimal 8GB RAM dan ruang disk kosong minimal 10GB untuk instalasi Android Studio yang optimal.

üçé XCode (Swift)

1Download XCode

Buka App Store di macOS, cari "Xcode" dan klik tombol "Get" atau "Download" untuk menginstall.

2Membuka XCode Pertama Kali

Setelah instalasi selesai, buka XCode dari Applications. Setujui license agreement dan tunggu instalasi additional components.

3Membuat Project Baru

Klik "Create a new Xcode project", pilih "App" di bagian iOS, lalu klik Next. Isi detail project:

  • Product Name: HelloWorld
  • Interface: SwiftUI
  • Language: Swift

4Project HelloWorld Berhasil Dibuat

Project HelloWorld telah berhasil dibuat. Anda dapat melihat preview aplikasi di sebelah kanan dengan SwiftUI preview.

üí° Catatan: XCode hanya tersedia untuk macOS. Pastikan macOS Anda memiliki versi yang kompatibel dengan XCode versi terbaru.

‚öõÔ∏è React Native (Multiplatform)

1Cek Versi Node.js dan Java

Buka terminal/command prompt dan jalankan perintah untuk memverifikasi instalasi Node.js dan Java. Jika belum terinstall, install terlebih dahulu:

node --version
java -version

2Cek Routing ANDROID_HOME

Pastikan environment variable ANDROID_HOME sudah di-set dengan benar. Jalankan perintah berikut untuk memverifikasi:

echo $ANDROID_HOME
# atau di Windows:
echo %ANDROID_HOME%

3Init Project dengan NPX

Buat project React Native baru menggunakan npx:

npx react-native init HelloWorld

4Init dengan React Native Community CLI

Alternatif lain, gunakan React Native Community CLI untuk membuat project:

npx @react-native-community/cli init HelloWorld

5Menjalankan Project

Masuk ke direktori project dan jalankan aplikasi:

cd HelloWorld
npm run android

6Ubah Code dan Jalankan Ulang

Edit file App.tsx atau App.js, ubah text menjadi "Hello World", kemudian jalankan kembali aplikasi:

npm run android

Aplikasi akan ter-reload otomatis dengan perubahan yang telah dibuat.

üí° Catatan: Untuk menjalankan di Android, pastikan Android Studio sudah terinstall. Untuk iOS, Anda memerlukan macOS dengan XCode terinstall.

üê¶ Flutter (Multiplatform)

1Download Flutter SDK

Kunjungi https://flutter.dev dan download Flutter SDK sesuai dengan sistem operasi Anda.

2Ekstrak Flutter SDK

Ekstrak file zip Flutter ke lokasi yang diinginkan (misalnya: C:\src\flutter atau ~/development/flutter).

3Setup Environment dan Flutter Doctor

Tambahkan path folder flutter/bin ke environment variable PATH sistem Anda. Kemudian jalankan flutter doctor untuk memeriksa dependensi:

flutter doctor

Perintah ini akan memeriksa dependensi yang diperlukan dan memberikan instruksi jika ada yang kurang.

4Install Extension VS Code

Buka VS Code dan install extension Flutter dan Dart dari marketplace untuk mendapatkan fitur development yang lebih baik.

5Flutter Create Hello World

Buat project Flutter baru dengan perintah:

flutter create hello_world

6Ubah main.dart dan Flutter Run

Buka file lib/main.dart, ubah kode sesuai kebutuhan, kemudian jalankan aplikasi dengan perintah:

cd hello_world
flutter run

7Aplikasi Berhasil Berjalan

Aplikasi Flutter berhasil berjalan di emulator atau device yang terhubung. Anda dapat melihat hasil perubahan secara real-time dengan hot reload.

üí° Catatan: Flutter mendukung hot reload yang memudahkan development. Pastikan emulator atau device sudah terhubung sebelum menjalankan aplikasi.

üî∑ Kotlin Multiplatform Mobile (KMM)

1Buka Plugin Android Studio

KMM memerlukan Android Studio sebagai IDE utama. Buka Android Studio, masuk ke Settings/Preferences ‚Üí Plugins, cari "Kotlin Multiplatform" dan install plugin tersebut.

2Restart IDE

Setelah plugin terinstall, restart Android Studio untuk mengaktifkan plugin.

3New Project - Pilih Kotlin Multiplatform

Klik "New Project", pilih "Kotlin Multiplatform App", lalu klik Next.

4Setup Project Name dan Platform

Isi detail project:

  • Name: Hello_World_kmm
  • Package name: com.example.hello_world_kmm
  • Platform: Pilih Android dan iOS

5Tunggu Gradle Selesai

Project akan menampilkan struktur multiplatform. Tunggu hingga proses Gradle build selesai sampai muncul composeApp di container kiri.

6Buka File App.kt

Navigasi ke composeApp > commonMain > kotlin > com.example.hello_world_kmm > App.kt untuk melihat kode shared.

7Ubah Text Menjadi "Hello World KMM"

Edit file App.kt, ubah Text menjadi:

Text("Hello World KMM")

Kemudian jalankan aplikasi untuk melihat perubahan.

üí° Catatan: Untuk menjalankan aplikasi iOS dari KMM, Anda memerlukan macOS dengan XCode terinstall. Anda dapat membuka modul iosApp di XCode.
